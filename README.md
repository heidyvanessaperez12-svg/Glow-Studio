# Glow Studio

Proyecto web desarrollado como práctica de diseño y desarrollo web.

## Tecnologías utilizadas
- HTML5
- CSS3
- Diseño responsive

## Descripción
Sitio web para un estudio de uñas que incluye:
- Sección principal (hero)
- Servicios
- Galería de diseños
- Ubicación con mapa
- Botón de contacto por WhatsApp
